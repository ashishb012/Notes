# Git Commands

## Git basic Commands

### Git Initialization

```bash
git init
```

### Git Add Files

```bash
git add -A
```

### Git Branch

```bash
git branch -M main
```

### Git Commit

```bash
git commit -m "commit message"
```

### Git remote add

```bash
git remote add origin https://github.com/username/reponame.git
```

### Git Push

```bash
git push origin main
```
